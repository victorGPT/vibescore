# Milestones

## M1 - Requirements & Acceptance
- Entry criteria: Design approved; repo_root_hash decision confirmed.
- Exit criteria: Requirements + acceptance criteria written.
- Required artifacts: requirements-analysis.md, acceptance-criteria.md.

## M2 - OpenSpec Proposal (if applicable)
- Entry criteria: M1 complete.
- Exit criteria: OpenSpec proposal/design/specs/tasks updated.
- Required artifacts: proposal.md, design.md, specs/**, tasks.md.

## M3 - Unit Test Coverage
- Entry criteria: M2 complete.
- Exit criteria: Unit tests for gating/pending/purge/hash pass.
- Required artifacts: test updates + passing test output.

## M4 - Regression & Integration
- Entry criteria: M3 complete.
- Exit criteria: Integration tests for uploader/ingest pass; system totals unchanged.
- Required artifacts: test updates + run logs.

## M5 - Release & Monitoring
- Entry criteria: M4 complete.
- Exit criteria: Documentation updated; verification report completed.
- Required artifacts: verification-report.md, updated docs/plans/.
