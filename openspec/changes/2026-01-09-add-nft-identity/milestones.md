# Milestones

## M1 - Requirements & Acceptance
- Entry criteria: Scope confirmed, assumptions listed.
- Exit criteria: requirements-analysis + acceptance-criteria approved.
- Required artifacts: `requirements-analysis.md`, `acceptance-criteria.md`.

## M2 - OpenSpec Proposal (if applicable)
- Entry criteria: M1 approved.
- Exit criteria: OpenSpec proposal + deltas validated.
- Required artifacts: `proposal.md`, `tasks.md`, `specs/vibescore-tracker/spec.md`.

## M3 - Unit Test Coverage
- Entry criteria: Schema + contracts drafted.
- Exit criteria: Core unit tests passing (masking, voucher expiry, aggregation math).
- Required artifacts: Test reports + updated `test-strategy.md` evidence.

## M4 - Regression & Integration
- Entry criteria: Edge functions + UI implemented.
- Exit criteria: Integration tests green + regression statement recorded.
- Required artifacts: Integration logs + regression note.

## M5 - Release & Monitoring
- Entry criteria: Contract deployed to Base mainnet (or testnet stage complete).
- Exit criteria: End-to-end mint + public display verified; monitoring in place.
- Required artifacts: Deployment notes + verification report.
