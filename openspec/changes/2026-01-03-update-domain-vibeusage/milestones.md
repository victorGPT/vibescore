# Milestones

## M1 - Requirements & Acceptance
- Entry criteria: Scope confirmed; final-domain pending recorded.
- Exit criteria: Requirements + acceptance criteria authored.
- Required artifacts: `requirements-analysis.md`, `acceptance-criteria.md`.

## M2 - OpenSpec Proposal
- Entry criteria: M1 complete.
- Exit criteria: Proposal + tasks + module brief + spec deltas drafted.
- Required artifacts: `proposal.md`, `tasks.md`, `module-brief.md`, `specs/vibeusage-tracker/spec.md`.

## M3 - Implementation
- Entry criteria: Proposal approved and final domain confirmed.
- Exit criteria: Domain configuration + code/docs updates complete (no deploy until final domain confirmed).
- Required artifacts: code changes, updated docs, Vercel config notes.

## M4 - Verification
- Entry criteria: M3 complete.
- Exit criteria: Redirect checks + build metadata checks complete; regression statement recorded.
- Required artifacts: verification notes and commands.
