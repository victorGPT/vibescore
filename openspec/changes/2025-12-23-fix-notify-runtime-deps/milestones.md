# Milestones

## M1 - Requirements & Acceptance
- Entry criteria: 需求确认。
- Exit criteria: `requirements-analysis.md` + `acceptance-criteria.md` 完成。
- Required artifacts: 需求/验收文档。

## M2 - OpenSpec Proposal (if applicable)
- Entry criteria: M1 完成。
- Exit criteria: `proposal.md` + `tasks.md` + delta spec 完成。
- Required artifacts: OpenSpec 变更目录。

## M3 - Unit Test Coverage
- Entry criteria: M2 完成。
- Exit criteria: 依赖就绪判断可验证。
- Required artifacts: 测试或验证脚本。

## M4 - Regression & Integration
- Entry criteria: M3 完成。
- Exit criteria: 回归脚本通过，notify 链路验证通过。
- Required artifacts: 运行记录。

## M5 - Release & Monitoring
- Entry criteria: M4 完成。
- Exit criteria: 记录回归命令与结果。
- Required artifacts: 验证报告。
