# Milestones

## M1 - Requirements & Acceptance
- Entry criteria: change ID created.
- Exit criteria: requirements analysis + acceptance criteria drafted.
- Required artifacts: `requirements-analysis.md`, `acceptance-criteria.md`.

## M2 - OpenSpec Proposal (if applicable)
- Entry criteria: M1 complete.
- Exit criteria: proposal + spec delta validated.
- Required artifacts: `proposal.md`, `specs/vibeusage-tracker/spec.md`, `tasks.md`, `openspec validate --strict`.

## M3 - Unit Test Coverage
- Entry criteria: design approved.
- Exit criteria: pricing resolver tests added.
- Required artifacts: test files + results.

## M4 - Regression & Integration
- Entry criteria: endpoint changes implemented.
- Exit criteria: runbook or acceptance script executed.
- Required artifacts: verification report updates.

## M5 - Release & Monitoring
- Entry criteria: M4 complete.
- Exit criteria: migration applied + docs updated.
- Required artifacts: deployment notes.
