# Milestones

## M1 - Requirements & Acceptance
- Entry criteria: Request clarified, scope defined.
- Exit criteria: Requirements + acceptance criteria drafted and reviewed.
- Required artifacts: `requirements-analysis.md`, `acceptance-criteria.md`.

## M2 - OpenSpec Proposal
- Entry criteria: M1 completed.
- Exit criteria: Proposal, tasks, and spec deltas drafted; `openspec validate` passes.
- Required artifacts: `proposal.md`, `tasks.md`, `specs/vibeusage-tracker/spec.md`.

## M3 - Unit Test Coverage
- Entry criteria: Implementation approach chosen.
- Exit criteria: Unit tests for trimming/pagination pass.
- Required artifacts: new tests under `test/`.

## M4 - Regression & Integration
- Entry criteria: Unit tests green.
- Exit criteria: Regression test (pre-usage history) passes; dashboard verification steps documented.
- Required artifacts: verification steps in `verification-report.md`.

## M5 - Release & Monitoring
- Entry criteria: M4 completed, change approved.
- Exit criteria: Deployed build verified; evidence map updated.
- Required artifacts: updated `openspec/specs/vibeusage-tracker/evidence.md`.
