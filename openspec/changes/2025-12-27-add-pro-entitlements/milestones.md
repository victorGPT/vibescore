# Milestones

## M1 - Requirements & Acceptance
- Entry criteria: Problem statement agreed; cutoff/timezone locked.
- Exit criteria: `requirements-analysis.md`, `acceptance-criteria.md` complete.
- Required artifacts: requirements + acceptance docs.

## M2 - OpenSpec Proposal (if applicable)
- Entry criteria: M1 complete.
- Exit criteria: proposal, tasks, spec delta validated.
- Required artifacts: `proposal.md`, `tasks.md`, `specs/vibeusage-tracker/spec.md`.

## M3 - Unit Test Coverage
- Entry criteria: data model and helper interface decided.
- Exit criteria: unit tests for pro status computation pass.
- Required artifacts: unit tests + green run.

## M4 - Regression & Integration
- Entry criteria: endpoints implemented.
- Exit criteria: `node --test` suite passes with new integration tests.
- Required artifacts: test run output, verification report updates.

## M5 - Release & Monitoring
- Entry criteria: all tests pass; docs updated.
- Exit criteria: deployment + monitoring notes recorded.
- Required artifacts: freeze record (if required), changelog update (if needed).
